Testing Ace.js with PyScript

What is PyScript?

PyScript is a programming language that allows the execution of Python code in the web browser using modern web technologies such as Pyodide and WASM (Web Assembly). It is designed to be accessible to users of all levels, with many applications due to its connection with Python.

More information about PyScript here.

What is Ace.js?

Ace is a code editor that can be integrated into web pages using JavaScript. It includes features like syntax highlighting and error marking for over 120 programming languages. It is used in Cloud9 IDE and is the successor to Mozilla’s Skywriter (Bespin) project.

More information about Ace.js here.

What is a REPL?

A REPL (Read-Eval-Print Loop) is an interactive environment that allows you to write code, evaluate it, and see the results immediately in the same environment.

With Ace.js and PyScript, we can simulate a REPL. What we do is wait for the “Run” button to be pressed to execute the code in the editor as if it were a PyScript script.
